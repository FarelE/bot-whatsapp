## Example bot whatsaap with open AI

## Intalasi

### For PC

```
> git clone https://github.com/ahlulmukh/bot-whatsapp.git
> cd bot-whatsapp
> npm install
> Change api key https://beta.openai.com/account/api-keys in database/key.json
> npm start
```
